
## 🛠️ 故障排除

### 前端無法啟動
1. 確認 Flutter 路徑正確
2. 檢查 3000 端口是否被佔用
3. 嘗試運行 `啟動前端.bat`

### 後端無法啟動
1. 確認 Docker Desktop 正在運行
2. 檢查 8000 端口是否被佔用
3. 運行 `docker compose ps` 查看容器狀態

### 登入失敗
1. 確認後端服務正常（訪問 http://localhost:8000/health）
2. 檢查測試帳號資訊是否正確
3. 查看瀏覽器控制台錯誤訊息

## 📝 停止應用

### 停止所有服務
雙擊執行：`stop.bat`

### 手動停止
```bash
# 停止後端
cd backend
docker compose down

# 停止前端
# 在運行 Flutter 的 CMD 視窗按 'q' 鍵
```
