###  1. Bug 修復（100%完成）

#### Bug #1: AI 建議無法生成 ✅
**問題**：用戶有自選股但AI建議頁面顯示「尚無 AI 建議」
**原因**：
- KeyError: 'low' - FinMind API 使用 `max`/`min` 而非 `high`/`low`
- KeyError: 'Foreign_Investor_buy' - 三大法人數據格式不符

**修復**：
- 添加欄位名稱標準化 ([ai_suggestion_service.py:32-37](backend/app/services/ai_suggestion_service.py#L32-L37))
- 重寫籌碼面分析函數 ([ai_suggestion_service.py:98-144](backend/app/services/ai_suggestion_service.py#L98-L144))

#### Bug #2: AI 建議價格型別錯誤 ✅
**問題**：TypeError: `"1820.00" type 'String' is not a subtype of type 'num'`
**原因**：後端 Decimal 序列化為字串而非數字

**修復**：
- 添加 JSON 編碼器 ([schemas/ai.py:29-31](backend/app/schemas/ai.py#L29-L31))
```python
json_encoders = {
    Decimal: lambda v: float(v) if v is not None else None
}
```

#### Bug #3: 股票搜尋支援完整驗證 ✅
**狀態**：後端已完整支援代號和名稱搜尋
**驗證**：測試通過，可搜尋「2330」、「台積電」、「鴻海」等

## API 端點總覽

### 股票相關
- `GET /api/stocks/search?q={query}` - 搜尋股票（代號或名稱）
- `GET /api/stocks/{stock_id}` - 取得股票詳情
- `GET /api/stocks/{stock_id}/price` - 取得即時報價
- `GET /api/stocks/{stock_id}/history?days={days}` ✨ **新增** - 取得歷史K線數據

### AI 相關
- `GET /api/ai/suggestions` - 取得所有AI建議
- `GET /api/ai/suggestions/{stock_id}` - 取得單一股票AI建議
- `POST /api/ai/chat` - AI 問答

### 自選股相關
- `GET /api/watchlist` - 取得自選股列表
- `POST /api/watchlist` - 添加自選股
- `DELETE /api/watchlist/{stock_id}` - 移除自選股

## 測試建議

### 1. K線圖測試 🧪
1. 進入「自選股」頁面
2. 點擊任一股票（如 2330 台積電）
3. 切換到「K線」Tab
4. 確認顯示：
   - ✅ 60日蠟燭圖（紅漲綠跌）
   - ✅ MA5、MA10、MA20 三條均線
   - ✅ 下方成交量柱狀圖
   - ✅ 圖例說明

### 2. AI 建議測試 🧪
1. 進入「AI 建議」頁面
2. 點擊「重新整理」按鈕
3. 等待 30-60 秒（首次生成）
4. 確認AI建議卡片包含：
   - ✅ 建議類型（BUY/HOLD/SELL）
   - ✅ 信心度（0-100%）
   - ✅ 進場價位區間 ✨
   - ✅ 三個停利目標 ✨
   - ✅ 停損價
   - ✨ 風險等級標籤
   - ✅ 操作週期 ✨
   - ✅ 預期漲跌幅 ✨
   - ✅ 詳細分析理由（200-300字）
   - ✅ 關鍵因素列表

### 3. 股票搜尋測試 🧪
1. 點擊自選股頁面的「＋」按鈕
2. 測試搜尋：
   - 輸入「台積電」 → 應顯示 2330
   - 輸入「2330」 → 應顯示台積電
   - 輸入「鴻」 → 應顯示鴻海等相關股票
   - 輸入「23」 → 應顯示所有 23XX 開頭的股票

### 4. 技術指標測試 🧪
1. 後端測試：查看 AI 建議生成的 reasoning
2. 確認包含技術指標分析：
   - ✅ MACD 信號（金叉/死叉）
   - ✅ RSI 超買超賣（70/30）
   - ✅ KD 指標與黃金交叉
   - ✅ 布林通道位置
   - ✅ 威廉指標
   - ✅ 均線排列
