## 🔧 技術亮點

### 1. 架構設計
```
前端 (Flutter Web)
    ↓ HTTP/REST
後端 (FastAPI)
    ↓ SQLAlchemy
資料庫 (PostgreSQL)
    ↓ API Call
AI 服務 (Google Gemini)
```

### 2. 核心技術棧

**後端**
- FastAPI 0.109.0
- PostgreSQL 15
- SQLAlchemy 2.0
- Google Generative AI 0.8.3

- Passlib + Bcrypt

**前端**
- Flutter 3.0+
- Provider 狀態管理
- HTTP 客戶端
- Secure Storage
