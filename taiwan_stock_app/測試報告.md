## 已修復的問題

### 問題 1: JWT Token 驗證失敗
**原因**: Token 的 sub 欄位為整數，jose 函式庫要求字串
**解決**: 修改為字串格式
**文件**: `backend/app/routers/auth.py:34`

### 問題 2: GOOGLE_API_KEY 未載入
**原因**: docker-compose.yml 未配置 env_file
**解決**: 添加 `env_file: .env`
**文件**: `backend/docker-compose.yml:24-25`

### 問題 3: google-generativeai 版本過舊
**原因**: 0.3.2 不支援新版 Gemini 模型
**解決**: 升級到 0.8.3
**文件**: `backend/requirements.txt:12`

### 問題 4: API 配額超過
**原因**: 免費配額限制
**解決**: 更換 API Key，使用 gemini-2.5-flash
**配置**: `backend/.env:7,13`

### 問題 5: WatchlistItem price 為字串
**原因**: Decimal 類型序列化為字串
**解決**: 改為 float 類型
**文件**: `backend/app/schemas/watchlist.py:18-19`

### 問題 6: 前端無法解析 price
**原因**: 未處理字串類型的價格
**解決**: 添加類型轉換和 null 檢查
**文件**: `frontend/lib/models/watchlist_item.dart:18-27`
